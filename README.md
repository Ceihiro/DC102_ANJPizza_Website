# AnJ Pizza Website
**Midterm Project | DC102 Web Development**

## About
A public website and in-store ordering system designed
for AnJ Pizza, a homemade pizza shop in Quezon City, Philippines.

## Pages
- `index.html` — Public website
- `admin.html` — Admin panel

## Features
- Public landing page with menu, about, gallery, and contact
- In-store ordering kiosk protected by an order code
- Admin login system
- Admin can manage pizza menu and add-ons
- Admin can view and update order status
- Admin can change the order code and password

## Tech Stack
- HTML
- CSS
- JavaScript

## Note
No database yet. Data is hardcoded for design demo purposes.
Database integration will be added in a future version.

## Design Reference
Design layout and structure inspired by 
Domino's Pizza PH (dominospizza.ph)
Adapted for a small homemade pizza shop.
